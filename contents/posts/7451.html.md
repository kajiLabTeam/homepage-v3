# 追加 / 編集の詳細
## 投稿日の変更
`#date:yyyy-mm-dd` タグで投稿日を変更できます

例) `#date:2005-01-05`

## ~id の指定~ (非推奨)
`#id:{n}` タグで URL の suffix を指定できます
例) `#id:123` → `/post/2020-01-01_123`

これは旧版の HUGO で同じ投稿日に suffix が付く仕様に合わせるための機能です
2025年4月10日以前に作成された記事のみ有効です
2025年4月10日以降はesaの番号が suffix になります
